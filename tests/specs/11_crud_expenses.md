# Expenses テスト仕様書
---

## TC-EXP-001 新規作成

| 項目 | 内容 |
|------|------|
| 手順 | 1. 「経費」一覧ページへ移動する<br>2. 「新規作成」ボタンをクリックする<br>3. タイトル（title）に「顧客訪問 タクシー代」と入力する<br>4. 金額（amount）に「3500」と入力する<br>5. カテゴリ（category）に「交通費」を選択する<br>6. 経費日（expense_date）を入力する<br>7. 関連取引先（account）を選択する<br>8. 関連担当者（contact）を選択する<br>9. 関連商談（opportunity）を選択する<br>10. 備考（notes）に補足情報を入力する<br>11. 「保存」ボタンをクリックする |
| 期待結果 | 経費レコードが正常に作成され、一覧ページにタイトル・金額・カテゴリ・経費日が表示される |

## TC-EXP-002 カテゴリ選択

| 項目 | 内容 |
|------|------|
| 手順 | 1. 「新規作成」ボタンをクリックする<br>2. カテゴリドロップダウンを開く<br>3. 「交通費」「接待費」「消耗品」「その他」の4選択肢が表示されることを確認する<br>4. それぞれのカテゴリで経費レコードを1件ずつ作成する |
| 期待結果 | カテゴリドロップダウンに「交通費」「接待費」「消耗品」「その他」の4種類が表示される。それぞれのカテゴリで正常に保存できる |

## TC-EXP-003 詳細ページ表示

| 項目 | 内容 |
|------|------|
| 前提 | 少なくとも1件の経費レコードが作成済みであること |
| 手順 | 1. 経費一覧ページへ移動する<br>2. 任意の経費レコードのタイトルリンクをクリックする |
| 期待結果 | 経費詳細ページが表示され、タイトル・金額・カテゴリ・経費日・関連取引先・担当者・商談・備考がすべて正しく表示される |

## TC-EXP-004 編集

| 項目 | 内容 |
|------|------|
| 前提 | 少なくとも1件の経費レコードが作成済みであること |
| 手順 | 1. 経費詳細ページを開く<br>2. 「編集」ボタンをクリックする<br>3. 金額を変更する<br>4. カテゴリを変更する<br>5. 備考を更新する<br>6. 「保存」ボタンをクリックする |
| 期待結果 | 変更内容が正常に保存され、詳細ページに更新後の値が表示される |

---

## TC-EXP-005 削除

| 項目 | 内容 |
|------|------|
| 前提 | 少なくとも1件の経費レコードが作成済みであること |
| 手順 | 1. 経費詳細ページを開く<br>2. 「削除」ボタンをクリックする<br>3. 確認ダイアログで「OK」をクリックする |
| 期待結果 | 経費レコードが削除され、一覧ページに該当レコードが表示されなくなる |

## TC-EXP-006 月別集計表示

| 項目 | 内容 |
|------|------|
| 前提 | 複数の月・複数のカテゴリにまたがる経費レコードが作成済みであること |
| 手順 | 1. 経費一覧ページへ移動する<br>2. 一覧の表示形式・集計セクションを確認する |
| 期待結果 | 経費が経費日または月ごとにグループ化されて表示される。カテゴリ別の内訳または合計金額が確認できる |
