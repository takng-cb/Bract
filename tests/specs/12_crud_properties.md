# Properties テスト仕様書
---

## TC-PRP-001 新規作成（基本情報）

| 項目 | 内容 |
|------|------|
| 手順 | 1. 「物件」一覧ページへ移動する<br>2. 「新規作成」ボタンをクリックする<br>3. 取引種別（transaction_type）に「売買」を選択する<br>4. ステータス（status）を選択する<br>5. 所在地（address）に「東京都千代田区〇〇1-2-3」と入力する<br>6. その他の基本情報を入力する<br>7. 「保存」ボタンをクリックする |
| 期待結果 | 物件レコードが正常に作成され、一覧ページに取引種別・ステータス・所在地が表示される。取引種別「賃貸」でも同様に作成できる |

---

## TC-PRP-002 土地登記情報入力

| 項目 | 内容 |
|------|------|
| 前提 | 物件レコードが1件作成済みであること（または新規作成フォームを開いた状態） |
| 手順 | 1. 物件の新規作成または編集フォームを開く<br>2. 土地登記情報セクションに移動する<br>3. 不動産番号（fudosan_number）を入力する<br>4. 地番（chiban）を入力する<br>5. 地目（chimoku）を入力する<br>6. 地積（area）を入力する<br>7. 所有者情報（owner info）を入力する<br>8. 「保存」ボタンをクリックする |
| 期待結果 | 土地の登記情報（不動産番号・地番・地目・地積・所有者情報）が正常に保存される |

---

## TC-PRP-003 建物登記情報入力

| 項目 | 内容 |
|------|------|
| 前提 | 物件レコードが1件作成済みであること（または新規作成フォームを開いた状態） |
| 手順 | 1. 物件の新規作成または編集フォームを開く<br>2. 建物登記情報セクションに移動する<br>3. 不動産番号（fudosan_number）を入力する<br>4. 家屋番号（kaoku）を入力する<br>5. 構造（structure）を入力する<br>6. 床面積（floor_area）を入力する<br>7. 「保存」ボタンをクリックする |
| 期待結果 | 建物の登記情報（不動産番号・家屋番号・構造・床面積）が正常に保存される |

## TC-PRP-004 担保・差押情報

| 項目 | 内容 |
|------|------|
| 前提 | 物件レコードが1件作成済みであること |
| 手順 | 1. 物件の新規作成または編集フォームを開く<br>2. 担保・差押情報セクションに移動する<br>3. 抵当権（lien）の詳細を入力する<br>4. 差押（seizure）の詳細を入力する<br>5. 「保存」ボタンをクリックする |
| 期待結果 | 担保・差押情報が正常に保存される。詳細ページで該当情報が表示される |

---

## TC-PRP-005 詳細ページ表示

| 項目 | 内容 |
|------|------|
| 前提 | 土地登記情報・建物登記情報・担保差押情報がすべて入力された物件レコードが作成済みであること |
| 手順 | 1. 物件一覧ページへ移動する<br>2. 対象物件のリンクをクリックして詳細ページを開く<br>3. 土地登記情報セクションを確認する<br>4. 建物登記情報セクションを確認する<br>5. 担保・差押情報セクションを確認する |
| 期待結果 | 物件詳細ページに土地登記情報（不動産番号・地番・地目・地積・所有者情報）と建物登記情報（不動産番号・家屋番号・構造・床面積）および担保・差押情報がすべて正しく表示される |

## TC-PRP-006 編集

| 項目 | 内容 |
|------|------|
| 前提 | 少なくとも1件の物件レコードが作成済みであること |
| 手順 | 1. 物件詳細ページを開く<br>2. 「編集」ボタンをクリックする<br>3. 取引種別・ステータスを変更する<br>4. 土地の地積を変更する<br>5. 建物の構造を変更する<br>6. 「保存」ボタンをクリックする |
| 期待結果 | 変更内容が正常に保存され、詳細ページに更新後の値が表示される |

---

## TC-PRP-007 削除

| 項目 | 内容 |
|------|------|
| 前提 | 少なくとも1件の物件レコードが作成済みであること |
| 手順 | 1. 物件詳細ページを開く<br>2. 「削除」ボタンをクリックする<br>3. 確認ダイアログで「OK」をクリックする |
| 期待結果 | 物件レコードが削除され、一覧ページに該当レコードが表示されなくなる |
