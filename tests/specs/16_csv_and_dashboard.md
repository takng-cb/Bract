# CSVインポート／エクスポート・ダッシュボードテスト仕様書

## 対象機能
- CSVエクスポート
- CSVインポート（新規・更新・エラー）
- インポートログ
- ダッシュボード集計
- フォーキャスト

---

## TC-CSV-001 CSVエクスポート（取引先）

| 項目 | 内容 |
|------|------|
| 前提 | 取引先レコードが存在する |
| 手順 | 1. 取引先一覧を開く<br>2. 「CSVエクスポート」をクリックする<br>3. ダウンロードされたファイルを開く |
| 期待結果 | CSVファイルがダウンロードされ、ヘッダー行に取引先名・業種・電話番号などの列が含まれ、レコードの件数分の行が存在する |

---

## TC-CSV-002 CSVインポート（新規）

| 項目 | 内容 |
|------|------|
| 前提 | 編集者以上でログイン済み、正しいフォーマットのCSVファイルを用意している |
| 手順 | 1. 取引先一覧の「CSVインポート」をクリックする<br>2. 新規レコードのみを含むCSVファイルを選択する<br>3. 列マッピングを確認して「インポート実行」をクリックする |
| 期待結果 | CSVの行数分のレコードが新規作成され、一覧に表示される |

---

## TC-CSV-003 CSVインポート（更新）

| 項目 | 内容 |
|------|------|
| 前提 | 既存レコードのIDを含むCSVファイルを用意している |
| 手順 | 1. 「CSVインポート」をクリックする<br>2. 既存IDを含むCSVファイルを選択する<br>3. 「インポート実行」をクリックする |
| 期待結果 | 既存IDと一致するレコードが更新され、新規IDの行は新規作成される |

---

## TC-CSV-004 CSVフォーマットエラー

| 項目 | 内容 |
|------|------|
| 前提 | 必須列が欠けているCSVファイルを用意している |
| 手順 | 1. 「CSVインポート」をクリックする<br>2. 不正なフォーマットのCSVファイルを選択する<br>3. 「インポート実行」をクリックする |
| 期待結果 | エラーメッセージが表示され、レコードはインポートされない |

---

## TC-CSV-005 編集権限なしのインポート不可

| 項目 | 内容 |
|------|------|
| 前提 | `viewer` ロールのユーザーとしてログイン済み |
| 手順 | 1. 取引先一覧を開く |
| 期待結果 | 「CSVインポート」ボタンが表示されない |

---

## TC-CSV-006 インポートログ確認

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み、過去にCSVインポートを実施済み |
| 手順 | 1. `/admin/import-logs` を開く |
| 期待結果 | インポートログ一覧に実行日時・対象オブジェクト・成功件数・エラー件数が表示される |

---

## TC-DASH-001 ダッシュボード表示

| 項目 | 内容 |
|------|------|
| 前提 | 各オブジェクトにレコードが存在する |
| 手順 | 1. `/dashboard` を開く |
| 期待結果 | 取引先・取引先担当者・商談・活動・タスクの件数がそれぞれのカードに表示される |

---

## TC-DASH-002 フォーキャスト表示

| 項目 | 内容 |
|------|------|
| 前提 | 商談レコードにステージと金額が設定されている |
| 手順 | 1. `/forecast` を開く |
| 期待結果 | 月別・ステージ別のパイプライン金額が表示される |

## TC-DASH-003 フォーキャスト年月切り替え

| 項目 | 内容 |
|------|------|
| 前提 | `/forecast` を開いている |

| 期待結果 | 選択した年月のフォーキャストデータに表示が更新される |
