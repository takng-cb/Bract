# カスタムオブジェクトテスト仕様書

## 対象機能
- カスタムオブジェクトの作成・削除
- カスタムフィールドの追加・並び替え・削除
- カスタムレコードの作成・編集・削除
- ナビゲーション表示

---

## TC-COBJ-001 カスタムオブジェクト作成

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み |
| 手順 | 1. `/admin/objects/new` を開く<br>2. `api_name`（英数字・アンダースコア）を入力する<br>3. ラベル（表示名）を入力する<br>4. アイコンを選択する<br>5. 「作成」をクリックする |
| 期待結果 | カスタムオブジェクトが作成され、`/admin/objects` の一覧に表示される |

---

## TC-COBJ-002 カスタムフィールド追加（テキスト）

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトが作成済み |
| 手順 | 1. 対象オブジェクトの設定画面を開く<br>2. 「フィールドを追加」をクリックする<br>3. フィールド種別に「テキスト」を選択する<br>4. フィールド名とラベルを入力する<br>5. 「保存」をクリックする |
| 期待結果 | テキスト型フィールドがオブジェクトに追加され、フィールド一覧に表示される |

---

## TC-COBJ-003 カスタムフィールド追加（数値／日付／選択肢／真偽値）

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトが作成済み |
| 手順 | 1. 「フィールドを追加」をクリックする<br>2. フィールド種別として「数値」「日付」「選択肢」「真偽値」のいずれかを選択する<br>3. 必要な設定（選択肢の場合は選択肢リスト）を入力する<br>4. 「保存」をクリックする |
| 期待結果 | 各フィールド種別が正しく追加され、レコード作成フォームに対応する入力 UI が表示される |

---

## TC-COBJ-004 フィールド順序変更

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトにフィールドが2件以上存在する |
| 手順 | 1. オブジェクト設定画面のフィールド一覧を開く<br>2. フィールドをドラッグ＆ドロップで並び替える<br>3. 「保存」をクリックする |
| 期待結果 | レコード作成・編集フォームおよび詳細画面でフィールドが新しい順序で表示される |

---

## TC-COBJ-005 フィールド削除

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトにフィールドが存在する |
| 手順 | 1. 削除対象フィールドの「削除」をクリックする<br>2. 確認ダイアログで「削除する」をクリックする |
| 期待結果 | フィールドが一覧から消え、レコードのフォームおよび詳細画面からも項目が消える |

---

## TC-COBJ-006 オブジェクトのナビ表示

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトが作成済み |
| 手順 | 1. オブジェクト設定画面で `nav_enabled` を有効にする<br>2. 設定を保存する<br>3. サイドバーを確認する |
| 期待結果 | サイドバーにカスタムオブジェクトのリンクが表示される |

---

## TC-COBJ-007 カスタムレコード作成

| 項目 | 内容 |
|------|------|
| 前提 | カスタムオブジェクトにフィールドが定義済みで、編集者以上でログイン済み |
| 手順 | 1. カスタムオブジェクトの一覧ページを開く<br>2. 「新規作成」をクリックする<br>3. 各フィールドに値を入力する<br>4. 「保存」をクリックする |
| 期待結果 | 新しいレコードが作成され、一覧に表示される |

---

## TC-COBJ-008 カスタムレコード編集

| 項目 | 内容 |
|------|------|
| 前提 | カスタムレコードが存在する |
| 手順 | 1. レコードの詳細ページを開く<br>2. 「編集」をクリックする<br>3. フィールドの値を変更する<br>4. 「保存」をクリックする |
| 期待結果 | 変更が保存され、詳細ページに更新後の値が表示される |

---

## TC-COBJ-009 カスタムレコード削除

| 項目 | 内容 |
|------|------|
| 前提 | カスタムレコードが存在する |
| 手順 | 1. レコードの詳細ページを開く<br>2. 「削除」をクリックする<br>3. 確認ダイアログで「削除する」をクリックする |
| 期待結果 | レコードが削除され、一覧に表示されなくなる |

---

## TC-COBJ-010 カスタムオブジェクト削除

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み、対象カスタムオブジェクトが存在する |
| 手順 | 1. `/admin/objects` で対象オブジェクトの「削除」をクリックする<br>2. 確認ダイアログで「削除する」をクリックする |
| 期待結果 | オブジェクト定義・関連フィールド・関連レコードがすべて削除され、一覧から消える。サイドバーからも項目が消える |
