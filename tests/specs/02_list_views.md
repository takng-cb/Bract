# リストビュー テスト仕様書

## 対象オブジェクト
取引先 / 担当者 / 商談 / 活動 / 経費 / タスク / 物件・商品

---

## TC-LIST-001 PC テーブル表示

| 項目 | 内容 |
|------|------|
| 手順 | 1. PC（幅 768px 以上）で各リストページを開く |
| 期待結果 | テーブルが表示される。レコードが存在する場合に行が表示される |
| 対象全ページ | accounts / contacts / opportunities / activities / expenses / tasks / properties |

---

## TC-LIST-002 モバイル カード表示

| 項目 | 内容 |
|------|------|
| 手順 | 1. スマホ（幅 767px 以下）で各リストページを開く |
| 期待結果 | カード形式のリストが表示される（テーブルは非表示） |
| 注意 | グルーピング中はカードが非表示になるのは仕様 |

---

## TC-LIST-003 空状態の表示

| 項目 | 内容 |
|------|------|
| 手順 | 1. レコードが0件のオブジェクトのリストページを開く |
| 期待結果 | 「まだありません」などの空状態メッセージが表示される |

---

## TC-LIST-004 ページネーション

| 項目 | 内容 |
|------|------|
| 前提 | 21件以上のレコードが存在する |
| 手順 | 1. リストページを開く<br>2. ページネーションの「次へ」をクリック |
| 期待結果 | 2ページ目が表示される（1ページ=20件） |

---

## TC-LIST-005 フィルター機能

| 項目 | 内容 |
|------|------|
| 手順 | 1. 「絞り込み」ボタンをクリック<br>2. 条件を追加して適用 |
| 期待結果 | 条件に一致するレコードのみ表示。「絞り込み中」ラベルが表示 |

---

## TC-LIST-006 フィルタークリア

| 項目 | 内容 |
|------|------|
| 手順 | 1. フィルター中に「絞り込みをクリア」をクリック |
| 期待結果 | 全件表示に戻る |

---

## TC-LIST-007 CSV エクスポート

| 項目 | 内容 |
|------|------|
| 手順 | 1. リストページで「エクスポート」ボタンをクリック |
| 期待結果 | CSV ファイルがダウンロードされる |

---

## TC-LIST-008 CSV インポート（編集権限あり）

| 項目 | 内容 |
|------|------|
| 前提 | 編集権限のあるユーザーでログイン |
| 手順 | 1. 「インポート」ボタンをクリック<br>2. 正しいフォーマットの CSV を選択 |
| 期待結果 | レコードがインポートされる |

---

## TC-LIST-009 列表示設定

| 項目 | 内容 |
|------|------|
| 手順 | 1. 「列」ボタンをクリック<br>2. 表示する列のチェックを変更 |
| 期待結果 | テーブルの表示列が切り替わる |
