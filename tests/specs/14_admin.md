# 管理機能テスト仕様書

## 対象機能
- ユーザー管理（一覧・ロール変更）
- アクセス制限
- システム設定
- ナビゲーション順序
- インポートログ
- 危険操作（全データ削除）

---

## TC-ADM-001 ユーザー一覧

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み |
| 手順 | 1. `/admin/users` を開く |
| 期待結果 | システムに登録されているすべてのユーザーがリスト表示される（メールアドレス・ロール・登録日などを含む） |

---

## TC-ADM-002 ロール変更

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み、対象ユーザーが存在する |
| 手順 | 1. `/admin/users` を開く<br>2. 対象ユーザーのロール欄をクリックする<br>3. `admin` / `editor` / `viewer` のいずれかに変更する<br>4. 変更を保存する |
| 期待結果 | ユーザーのロールが更新され、一覧に新しいロールが反映される |

---

## TC-ADM-003 一般ユーザーのアクセス制限

| 項目 | 内容 |
|------|------|
| 前提 | `viewer` ロールのユーザーとしてログイン済み |
| 手順 | 1. `/admin/users` へ直接アクセスする |
| 期待結果 | アクセスが拒否され、エラーページまたはダッシュボードへリダイレクトされる |

---

## TC-ADM-004 システム設定（会社名）

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み |
| 手順 | 1. `/settings` を開く<br>2. 会社名フィールドを新しい名前に変更する<br>3. 「保存」をクリックする |
| 期待結果 | 設定が保存され、ページ再読み込み後も変更した会社名が表示される |

---

## TC-ADM-005 ナビゲーション順序変更（システム）

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み |
| 手順 | 1. `/settings` のナビゲーション設定セクションを開く<br>2. ドラッグ＆ドロップでナビ項目の順序を変更する<br>3. 「保存」をクリックする |
| 期待結果 | システム全体のデフォルトナビゲーション順序が更新され、他のユーザーにも反映される |

---

## TC-ADM-006 ナビゲーション順序変更（個人）

| 項目 | 内容 |
|------|------|
| 前提 | 任意のロールでログイン済み |
| 手順 | 1. 個人設定のナビゲーションセクションを開く<br>2. ドラッグ＆ドロップで個人用ナビ順序を変更する<br>3. 「保存」をクリックする |
| 期待結果 | 自分のアカウントでのみナビゲーション順序が上書きされ、他のユーザーには影響しない |

---

## TC-ADM-007 インポートログ

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み、過去にCSVインポートを実施済み |
| 手順 | 1. `/admin/import-logs` を開く |
| 期待結果 | インポート履歴が一覧表示され、各行に実行日時・対象オブジェクト・成功件数・エラー件数が表示される |

---

## TC-ADM-008 全データ削除（危険操作）

| 項目 | 内容 |
|------|------|
| 前提 | 管理者としてログイン済み |
| 手順 | 1. `/settings` のデンジャーゾーンセクションを開く<br>2. 「全データ削除」ボタンをクリックする<br>3. 確認ダイアログに指定文字列を入力する<br>4. 「削除する」をクリックする |
| 期待結果 | 全レコードが削除され、各オブジェクトの一覧が空になる |
