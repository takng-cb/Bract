# タグテスト仕様書

## 対象機能
- タグの作成・編集・削除
- レコードへのタグ付け・取り外し
- タグによるフィルタリング

---

## TC-TAG-001 タグ作成

| 項目 | 内容 |
|------|------|
| 前提 | 管理者または編集者としてログイン済み |
| 手順 | 1. `/settings` またはタグ管理画面を開く<br>2. 「タグを追加」をクリック<br>3. タグ名を入力する<br>4. カラーパレットから色を選択する<br>5. 「保存」をクリックする |
| 期待結果 | 新しいタグが一覧に表示され、指定した名前と色が反映されている |

---

## TC-TAG-002 タグ編集

| 項目 | 内容 |
|------|------|
| 前提 | 既存のタグが存在する |
| 手順 | 1. タグ管理画面で対象タグの「編集」をクリック<br>2. タグ名または色を変更する<br>3. 「保存」をクリックする |
| 期待結果 | タグの名前・色が更新され、そのタグが付いたすべてのレコードに変更が反映される |

---

## TC-TAG-003 タグ削除（未使用）

| 項目 | 内容 |
|------|------|
| 前提 | どのレコードにも付いていないタグが存在する |
| 手順 | 1. タグ管理画面で対象タグの「削除」をクリック<br>2. 確認ダイアログで「削除する」をクリックする |
| 期待結果 | タグが一覧から消える |

---

## TC-TAG-004 取引先へのタグ付け

| 項目 | 内容 |
|------|------|
| 前提 | 取引先レコードが存在し、タグが1件以上作成済み |
| 手順 | 1. 取引先の詳細ページを開く<br>2. タグセクションの「＋」または「タグを追加」をクリックする<br>3. 表示されたタグ候補から任意のタグを選択する |
| 期待結果 | 選択したタグが詳細ページのタグ欄に表示される |

---

## TC-TAG-005 タグの複数付け

| 項目 | 内容 |
|------|------|
| 前提 | タグが2件以上作成済みで、対象レコードが存在する |
| 手順 | 1. レコードの詳細ページを開く<br>2. タグ追加 UI から1つ目のタグを選択する<br>3. 続けて2つ目のタグを選択する |
| 期待結果 | 両方のタグが同一レコードに表示される |

---

## TC-TAG-006 タグ外し

| 項目 | 内容 |
|------|------|
| 前提 | タグが付いているレコードが存在する |
| 手順 | 1. レコードの詳細ページを開く<br>2. 取り外したいタグの「×」ボタンをクリックする |
| 期待結果 | そのタグがレコードから取り外され、タグ欄に表示されなくなる |

---

## TC-TAG-007 タグによるフィルター

| 項目 | 内容 |
|------|------|
| 前提 | 同じタグが付いたレコードが複数存在する |
| 手順 | 1. 取引先（または任意オブジェクト）の一覧を開く<br>2. フィルターメニューでタグを選択する<br>3. 絞り込みたいタグを指定する |
| 期待結果 | 指定したタグが付いたレコードだけが一覧に表示される |
