# PWA・モバイル テスト仕様書

---

## TC-PWA-001 Android Chrome インストールプロンプト

| 項目 | 内容 |
|------|------|
| 環境 | Android Chrome |
| 手順 | 1. `https://bract-crm.vercel.app` を開く<br>2. Chrome メニュー（⋮）を開く |
| 期待結果 | 「アプリをインストール」または「ホーム画面に追加」が「インストール」として表示される（「ショートカットを作成」ではない） |
| 補足 | 初回訪問直後は表示されないことがある（Chrome のエンゲージメント要件） |

## TC-PWA-002 PWA としての起動

| 項目 | 内容 |
|------|------|
| 前提 | TC-PWA-001 でインストール済み |
| 手順 | 1. ホーム画面のアイコンをタップ |
| 期待結果 | ブラウザのアドレスバーなしでスタンドアロン表示で起動する |

## TC-PWA-003 Web App Manifest の確認

| 項目 | 内容 |
|------|------|
| 手順 | 1. Chrome DevTools > Application > Manifest を確認 |
| 期待結果 | name, short_name, display: standalone, icons（192x192, 512x512）が表示される |

## TC-PWA-004 Service Worker の登録確認

|------|------|
