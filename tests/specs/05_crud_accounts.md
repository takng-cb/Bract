# 取引先 CRUD テスト仕様書
---

## TC-ACC-001 新規作成（正常）

| 項目 | 内容 |
|------|------|
| 前提 | ログイン済み、取引先一覧画面を表示している |
| 手順 | 1. 「新規作成」ボタンをクリックする<br>2. 取引先名（name）に「株式会社テスト」を入力する<br>3. 任意項目として業種（industry）、取引先種別（type）、電話番号（phone）、Webサイト（website）、住所（address）、年間売上（annual_revenue）、従業員数（employee_count）、ステータス（status）、説明（description）を入力する<br>4. 「保存」ボタンをクリックする |
| 期待結果 | 取引先が正常に作成され、詳細ページへリダイレクトされる。入力した全フィールドが正しく表示される |

---

## TC-ACC-002 新規作成（必須項目未入力）

| 項目 | 内容 |
|------|------|
| 前提 | ログイン済み、新規取引先作成フォームを表示している |
| 手順 | 1. 取引先名（name）を空のままにする<br>2. 「保存」ボタンをクリックする |
| 期待結果 | バリデーションエラーが表示され、「取引先名は必須です」などのエラーメッセージが name フィールドの近くに表示される。レコードは作成されない |

## TC-ACC-003 詳細ページ表示

| 項目 | 内容 |
|------|------|
| 前提 | 関連する担当者・商談・活動・タスク・添付ファイルが紐付いた取引先レコードが存在する |
| 手順 | 1. 取引先一覧から対象レコードをクリックして詳細ページを開く |
| 期待結果 | 詳細ページに以下のセクションがすべて表示される：基本情報、関連担当者、関連商談、活動履歴、タスク、添付ファイル、変更履歴 |

## TC-ACC-004 編集（正常）

| 項目 | 内容 |
|------|------|
| 前提 | 既存の取引先レコードの詳細ページを表示している |
| 手順 | 1. 「編集」ボタンをクリックする<br>2. 取引先名を「株式会社テスト（更新）」に変更する<br>3. 「保存」ボタンをクリックする |
| 期待結果 | 変更内容が保存され、詳細ページに更新後の値が表示される。変更履歴セクションに今回の変更が記録される |

---

## TC-ACC-005 ステータス変更

| 項目 | 内容 |
|------|------|
| 前提 | ステータスが「見込み」の取引先の詳細ページを表示している |
| 手順 | 1. ステータスフィールドのクイックアクション（ドロップダウンまたはボタン）をクリックする<br>2. 「有効」を選択する<br>3. 変更を確定する |
| 期待結果 | ステータスが「有効」に更新される。同様の操作で「無効」への変更も可能。変更履歴に変更前後の値が記録される |

---

## TC-ACC-006 削除

| 項目 | 内容 |
|------|------|
| 前提 | 既存の取引先レコードの詳細ページを表示している |
| 手順 | 1. 「削除」ボタンをクリックする<br>2. 確認ダイアログで「削除する」を選択する |
| 期待結果 | レコードが削除され、取引先一覧ページへリダイレクトされる。削除された取引先は一覧に表示されない |

---

## TC-ACC-007 関連担当者表示

| 項目 | 内容 |
|------|------|
| 前提 | 複数の担当者が紐付いた取引先レコードの詳細ページを表示している |
| 手順 | 1. 詳細ページの「関連担当者」セクションを確認する |
| 期待結果 | 紐付いているすべての担当者の氏名・役職・メールアドレスなどが一覧表示される。各担当者の行をクリックすると担当者詳細ページへ遷移する |

---

## TC-ACC-008 関連商談表示

| 項目 | 内容 |
|------|------|
| 前提 | 複数の商談が紐付いた取引先レコードの詳細ページを表示している |
| 手順 | 1. 詳細ページの「関連商談」セクションを確認する |
| 期待結果 | 紐付いているすべての商談の名称・ステージ・金額・完了予定日などが一覧表示される。各商談の行をクリックすると商談詳細ページへ遷移する |

---

## TC-ACC-009 添付ファイル アップロード

| 項目 | 内容 |
|------|------|
| 前提 | 既存の取引先レコードの詳細ページを表示している |
| 手順 | 1. 「添付ファイル」セクションの「アップロード」ボタンをクリックする<br>2. ファイル選択ダイアログでファイルを選択する（例：PDF、画像）<br>3. アップロードを確定する |
| 期待結果 | ファイルが正常にアップロードされ、添付ファイルセクションにファイル名・種類・アップロード日時が表示される。ファイル名をクリックするとダウンロードまたはプレビューが可能 |

## TC-ACC-010 変更履歴

| 項目 | 内容 |
|------|------|
| 前提 | 過去に編集された取引先レコードの詳細ページを表示している |
| 手順 | 1. 詳細ページの「変更履歴」セクションを確認する |
| 期待結果 | 変更ごとに、変更日時・変更者・変更フィールド名・変更前の値・変更後の値が一覧表示される |
