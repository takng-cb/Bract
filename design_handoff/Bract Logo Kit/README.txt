■ 使い方の要点
  ・基本はマーク＋ワードマーク（Geist Bold）の横組み。
  ・アプリ/favicon はサージュ地＋白抜きマーク。
  ・最小サイズ：アプリ40px / UI 24px / favicon 16px。
  ・縦横比変更・回転・影付け・ブランド外配色・改変は不可。
  ・追加サイズの PNG が必要な場合は各 SVG から書き出してください。
