# 個人データの取扱いに関する覚書 (DPA: Data Processing Agreement)

{{事業者名}}（以下「受託者」）と本サービスを契約する法人（以下「委託者」）は、{{サービス名}}（以下「本サービス」）の利用に伴い、委託者が受託者に取り扱わせる個人データ（以下「委託個人データ」）について、個人情報の保護に関する法律（以下「個情法」）第27条第5項第1号に基づく委託として、本覚書を締結します。

本覚書は、委託者が本サービスを契約した時点で、利用規約と一体として効力を生じます。

## 第1条（目的）

本覚書は、委託個人データの取扱いに関する委託者と受託者の責任分担および安全管理について定めることを目的とします。

## 第2条（委託の範囲）

1. **委託する業務**：本サービスを通じた委託個人データの保管、検索、表示、編集、バックアップ、削除等の処理
2. **委託個人データの種類**：委託者が本サービスに登録する取引先担当者・関係者の氏名、所属、連絡先、商談履歴等
3. **取扱期間**：本サービスの利用契約期間中

## 第3条（受託者の義務）

1. 受託者は、委託個人データを委託の目的の範囲内でのみ取り扱い、それ以外の目的に使用しません。
2. 受託者は、委託個人データに関し、個情法第23条（安全管理措置）に基づく必要かつ適切な措置を講じます。具体的措置は別表のとおりです。
3. 受託者は、自己の従業者に対し、委託個人データの取扱いに関する適切な監督を行います。

## 第4条（再委託）

1. 受託者は、委託個人データの取扱いの全部または一部を第三者に再委託する場合、委託者の同意を得るものとします。
2. 別添の業務委託先一覧（プライバシーポリシー第4項に列挙する Supabase、Vercel、Stripe 等）への再委託については、委託者は本覚書締結時点で同意したものとみなします。
3. 当該リストに変更がある場合、受託者は本サービス上の通知により委託者に告知するものとし、委託者が異議申立を行わない場合、変更に同意したものとみなします。
4. 受託者は、再委託先に対し、本覚書と同等以上の義務を課すものとします。

## 第5条（外国にある第三者への提供）

受託者は、委託個人データを外国（米国に所在する Vercel、Stripe 等）に所在する再委託先で取り扱わせる場合があります。詳細は、本サービスのプライバシーポリシー第5項のとおりです。委託者は、本覚書締結により、当該外国における委託個人データの取扱いに同意するものとします。

## 第6条（漏洩等の通知）

1. 受託者は、委託個人データの漏洩、滅失または毀損（以下「漏洩等」）を覚知した場合、覚知後速やかに（24 時間以内を目標に）委託者に通知します。
2. 通知には、漏洩等の概要、漏洩等が生じた個人データの項目、本人の数、原因、二次被害の有無、講じた措置を含めます。
3. 個情法第26条に基づく個人情報保護委員会への報告および本人通知は、原則として委託者（個人情報取扱事業者）の責任で行うものとし、受託者はこれに必要な情報提供を行います。

## 第7条（監査）

委託者は、受託者の委託個人データの取扱状況について、合理的な範囲で報告を求めることができます。受託者は、SOC 2 報告書、第三者認証取得状況、再委託先のセキュリティ証明書等の提示により、これに代えることができます。

## 第8条（終了時の取扱い）

1. 利用契約終了後、受託者は委託者の指示に従い、委託個人データを返却または削除します。
2. 委託者からの指示なく解約日から 30 日経過した場合、受託者は委託個人データを削除する権利を有します。
3. 法令により保存が義務付けられている記録は、保存期間経過後に削除します。

## 第9条（責任の分担）

1. 委託者（個人情報取扱事業者）は、本人からの開示・訂正・利用停止等の請求への対応について、最終的な責任を負います。
2. 受託者は、委託者からの要請に基づき、開示・訂正・削除等の作業を本サービス上の機能を通じて支援します。
3. 受託者の本覚書違反により委託者に損害が生じた場合、利用規約第13条（損害賠償の制限）が適用されます。

## 第10条（協議解決）

本覚書に定めのない事項または解釈に疑義が生じた場合、委託者と受託者は誠意をもって協議の上、解決するものとします。

## 別表：受託者の安全管理措置

| 区分 | 措置内容 |
|---|---|
| **組織的** | 個人情報取扱責任者の設置、取扱規程の整備、定期的な内部点検 |
| **人的** | 従事者への教育、秘密保持義務の誓約 |
| **物理的** | データセンター（Supabase 東京リージョン）の物理セキュリティに準拠 |
| **技術的** | アクセス制御（ロールベース・行レベル）、通信暗号化（TLS 1.2+）、保管時暗号化（AES-256）、不正アクセス監視、監査ログの保存（最低 90 日） |
| **外的環境** | 米国所在の再委託先について、個情法第28条に基づき制度を把握し、契約により安全管理を担保 |
