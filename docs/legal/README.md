# Bract CRM 法務書類テンプレート集

> ⚠️ **重要：これらは雛形です。施行前に必ず弁護士のレビューを受けてください。**
> 本テンプレートは一般的な SaaS 事業を想定した参考例であり、特定の法的助言ではありません。
> 業種・契約相手・個別事情によって追加・修正が必要です。

## プレースホルダ一覧

各文書中の `{{}}` で囲まれた箇所は、契約前に実値へ置換してください。

- `{{事業者名}}` … 例: 個人事業 → 屋号 + 代表者名 / 法人 → 株式会社○○
- `{{代表者名}}`
- `{{所在地}}` … 自宅住所を出したくない場合はバーチャルオフィス推奨
- `{{連絡先メール}}` … `support@` など
- `{{電話番号}}` … 特商法上「遅滞なく開示」明記で省略可
- `{{適格請求書発行事業者登録番号}}` … T + 13桁
- `{{管轄裁判所}}` … 例: 東京地方裁判所
- `{{サービス名}}` … Bract CRM
- `{{サービスURL}}` … 例: https://bract-crm.example.com
- `{{施行日}}` … YYYY 年 MM 月 DD 日
