> **対象**: 営業 / 経営判断 / 顧客提示資料の元ネタ

## 1. プラン体系

### スタータープラン（標準・初期顧客向け）

| 項目 | 金額 |
|---|---|
| **初期費用** | ¥100,000 – ¥150,000 |
| **基本料金（月額）** | **¥8,000** |
| **ユーザー単価（月額）** | **¥3,500 / user** |
| **対象規模** | 1 業種 / 1〜5 ユーザー想定（最大 20 ユーザーまで対応）|

#### 計算例（基本料金 + ユーザー単価）

| ユーザー数 | 計算 | 月額 |
|---|---|---|
| 1 | 8,000 + 3,500 × 1 | **¥11,500** |
| 2 | 8,000 + 3,500 × 2 | **¥15,000** |
| 3 | 8,000 + 3,500 × 3 | **¥18,500** |
| 5 | 8,000 + 3,500 × 5 | **¥25,500** |
| 10 | 8,000 + 3,500 × 10 | **¥43,000** |

#### 初期費用に含まれるもの

- アカウント開設・初期設定
- 顧客の業種に応じた業種オーバーレイ設定（auto-body / real-estate / staffing 等）
- 既存データ移行支援（最大 1,000 レコードまで）
- 管理者 1 名分の操作レクチャー（オンライン 2 時間）

#### 月額に含まれるもの

- すべての標準機能の利用
- メールサポート（営業時間内、48 時間以内に初回返信）
- 自動バックアップ（日次・7 日間保持）
- セキュリティアップデート
- 機能アップデート（追加料金なし）

## 5. アーリーアダプター価格（初期 10 社限定）

実績作りと顧客フィードバック獲得のため、以下の特別価格を提供。

### 価格

| 項目 | 金額 |
|---|---|
| **初期費用** | **¥50,000**（通常 ¥100,000-150,000 → 約 60% OFF） |
| **基本料金（月額）** | **¥5,000**（通常 ¥8,000） |
| **ユーザー単価（月額）** | **¥3,000 / user**（通常 ¥3,500） |

#### 計算例

| ユーザー数 | 月額 |
|---|---|
| 1 | ¥8,000 |
| 3 | ¥14,000 |
| 5 | **¥20,000** |

### 条件

- 改善要望の提出に協力（月 1 回程度の定例 MTG）
- 事例公開許諾（社名・業種・利用シーンの公開）
- 契約後 1 年は固定価格、**2 年目から通常価格にスライド**
- 1 年以内に解約された場合、初期費用の差額（¥50,000 - ¥150,000）を遡及請求

### 受付期間

最初の 10 社契約まで、または 2026 年 12 月末まで（早い方）

## 6. LTV（顧客生涯価値）シミュレーション

### 3 ユーザー / 3 年契約（通常価格）

```
初期費用     ¥120,000
月額         ¥18,500 × 36 ヶ月 = ¥666,000
─────────────────────────────────
合計（3 年）  ¥786,000
```

### 5 ユーザー / 3 年契約（通常価格）

```
初期費用     ¥150,000
月額         ¥25,500 × 36 ヶ月 = ¥918,000
─────────────────────────────────
合計（3 年）  ¥1,068,000
```

| 顧客内訳 | 年商 |
|---|---|
| 3 user 平均 × 10 社 | ¥2,220,000 + 初期 ¥1,200,000 = **¥3,420,000** |
| 5 user 平均 × 10 社 | ¥3,060,000 + 初期 ¥1,500,000 = **¥4,560,000** |
| 上記 + AI ライセンス 50% 採用 | **+¥300,000 / 年** |

```
3 user 平均 × 50 社 = 年商 ¥17,100,000
5 user 平均 × 50 社 = 年商 ¥22,800,000
```

## 8. 価格決定の前提

### 含まれる機能（Starter プラン）

- ✅ 取引先・人物・商談・活動・ToDo・経費管理
- ✅ カスタムオブジェクト・フィールド
- ✅ 権限管理（admin / editor / viewer）
- ✅ 監査ログ
- ✅ タグ・関連レコード
- ✅ CSV インポート / エクスポート
- ✅ 業種オーバーレイ（auto-body / real-estate / staffing から 1 業種）
- ✅ ダッシュボード・売上予測
- ✅ レスポンシブ Web UI
- ✅ 自動バックアップ（日次・7 日間保持）

### 含まれないもの（別料金 or 未実装）

- ❌ AI まとめ機能 → 追加ライセンス
- ❌ LINE 連携 → 追加ライセンス（実装後）
- ❌ 追加業種 → 追加ライセンス
- ❌ ネイティブモバイルアプリ → 未実装
- ❌ SSO / SAML 認証 → エンタープライズ向け（[#60](https://github.com/takng-cb/Bract-CRM/issues/60)）
- ❌ SCIM プロビジョニング → エンタープライズ向け（[#61](https://github.com/takng-cb/Bract-CRM/issues/61)）
- ❌ 国内データ保管 → エンタープライズ向け（[#62](https://github.com/takng-cb/Bract-CRM/issues/62)）
- ❌ 紙の契約書 → スタンダード 以上で提供予定（個別契約は要相談）

### セキュリティ・コンプライアンス

- ✅ Vercel (SOC 2 Type II 取得)
- ✅ Supabase Auth (SOC 2 取得)
- ✅ Neon Postgres (保管時暗号化)
- ✅ HTTPS 強制、RBAC、監査ログ
- ⚠ プライバシーマーク → 取得中（[#58](https://github.com/takng-cb/Bract-CRM/issues/58)、半年以内目標）
- ⚠ ISMS / SOC 2（自社）→ 検討中（[#59](https://github.com/takng-cb/Bract-CRM/issues/59)）

## 9. 改訂方針

- **既存契約の価格は契約期間中は固定**（インフレ等で値上げする場合は事前通知）
- **新規契約の価格は予告なく変更可能**
- 業界価格動向は **6 ヶ月ごとに見直し**
- 新機能追加に伴う価格改定は **半年前に既存顧客へ通知**

---

## 10. 営業時のチェックリスト

見積提示時の確認事項:

- [ ] 顧客の業種は対応済みか（auto-body / real-estate / staffing）
- [ ] ユーザー数（admin / editor / viewer 内訳）
- [ ] 必要なライセンス（AI / LINE / 追加業種）
- [ ] データ移行の規模（レコード数、形式）
- [ ] オンサイト研修の要否
- [ ] 契約期間（月払い / 年払い / 3 年契約）
- [ ] アーリーアダプター価格適用可否
- [ ] 個人情報保護法・インボイス制度等の要求事項
- [ ] エンタープライズ要件の有無（SSO / 国内保管 等）
