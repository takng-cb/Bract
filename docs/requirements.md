**バージョン**: 1.0  
**作成日**: 2026-05-06  
**対象システム**: https://bract-crm.vercel.app

---

## 1. システム概要

Bract CRM は不動産業・営業組織向けのウェブ型 CRM（顧客関係管理）システムである。  
Next.js (App Router) + Supabase (Auth / PostgreSQL) + Vercel でホストする SaaS アプリケーション。

### 1.1 目的

- 取引先・担当者・商談・活動・タスク・経費・物件の一元管理
- 営業進捗の可視化と予測
- チーム内の情報共有と権限管理

### 1.2 対象ユーザー

| ロール | 説明 |
|--------|------|
| 管理者（admin） | 全機能利用可。ユーザー管理・システム設定・カスタムオブジェクト定義 |
| 編集者（editor） | レコードの作成・編集・削除。CSV インポート。 |
| 閲覧者（viewer） | 参照のみ。作成・編集・削除・インポート不可。 |

## 3. 非機能要件

### 3.1 パフォーマンス

| 区分 | 要件 |
|------|------|
| ページ初期表示 | 3秒以内 |
| CRUD 操作 | 2秒以内にレスポンス |
| CSV エクスポート | 1,000件以内で5秒以内 |

### 3.2 対応環境

| 種別 | 対象 |
|------|------|
| ブラウザ（PC） | Chrome 最新版、Edge 最新版、Safari 最新版 |
| ブラウザ（モバイル） | Android Chrome 最新版、iOS Safari 最新版 |
| 画面幅 | 320px 〜（レスポンシブ対応） |
| PWA | Android Chrome でのホーム画面インストール対応 |

### 3.3 セキュリティ

| 区分 | 要件 |
|------|------|
| 認証 | Supabase Auth（JWT ベース） |
| API 保護 | 全 API エンドポイントで認証チェック |
| HTTPS | 全通信 TLS 1.2 以上 |
| 権限制御 | ロールによる機能アクセス制御（RBAC） |
| パスワード | bcrypt ハッシュ化（Supabase Auth 標準） |

### 3.4 可用性・信頼性

| 区分 | 要件 |
|------|------|
| ホスティング | Vercel（Hobby/Pro プラン） |
| データベース | Neon PostgreSQL（Supabase Cloud 互換） |
| バックアップ | Neon 自動バックアップ |
| SLA | Vercel/Neon の SLA に準拠 |

## 4. データモデル概要

```
users ─────┐
           ├── accounts ──── contacts
           │       └──────── opportunities ──── activities
           │                       └──────────── tasks
           │                                     expenses
           └── properties
               
tags ── taggables（polymorphic: accounts / contacts / opportunities / ...）

object_definitions ── field_definitions
                   └── custom_records ── custom_field_values

attachments（polymorphic: accounts / contacts / opportunities / activities）
change_logs（polymorphic）
import_logs
system_settings
saved_views
list_view_settings
user_preferences
```

## 6. 用語定義

| 用語 | 定義 |
|------|------|
| 取引先 | 取引対象の企業・組織 |
| 担当者 | 取引先に所属する人物、または個人顧客 |
