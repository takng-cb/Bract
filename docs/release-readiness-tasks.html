Vercel 環境変数の本番設定確認
#67 30分
▶

目的

AI 機能・LINE 連携などの feature flag が正しく設定されているか確認

手順

  1. Vercel プロジェクト → Settings → Environment Variables
  2. 以下を確認・設定:
    • DATABASE_URL (auto-body Neon)
    • NEXT_PUBLIC_INDUSTRY=auto-body
    • NEXT_PUBLIC_APP_URL (本番 URL)
    • SUPABASE_* 系
    • AI_FEATURE_ENABLED ← 契約に応じて
    • LINE_FEATURE_ENABLED ← 未実装なので未設定
  3. Production / Preview / Development の scope を適切に
