# 要件ログ（Requirements Log）

**追記専用。** 要件が出たら即ここに `REQ-NNNN` で記録する（漏れ防止のため判断せずまず追記）。
状態の流れ：`提案` → `合意` → `設計済` → `実装中` → `完了`（または `保留` / `却下`）。

書式：
```
### REQ-NNNN  〈短いタイトル〉
- 日付 / 出所（会話 / ブリーフ §x / Issue #y）
- 内容：何を満たしたいか
- 状態：（現在）
- 関連：ADR-xxxx / #Issue / spec:<module>#<節>
```
